== Crazy Piano 瘋狂鋼琴家

程式簡介：
Java寫的小遊戲，遊戲方式是會播放一小段旋律，玩家憑著自己的音感，並且彈出正確的音就可以過關。
一共有三個關卡：簡單、普通跟困難。

遊戲畫面：

https://i.imgur.com/spHraLe.png

使用套件：

1. Jfugue
2. Javafx

使用資源：

1. 背景圖片：插畫家-恩力君 https://www.facebook.com/ZNG.LIFE/?fref=ts
2. 音樂：惡之召使、千本櫻。
